# おまいらタイピング

```
　 ∧＿∧
　（　´∀｀）
　/　　　 ヽ
```

日本語タイピング練習ツール

![Demo](docs/demo.gif)

## 遊ぶ

**[https://zabio3.github.io/omaira/](https://zabio3.github.io/omaira/)**

## 特徴

- 複数のローマ字入力パターンに対応（shi/si など）
- 単語クリアで +2秒 延長
- タイピング分析（ミス箇所、苦手キー）

## 開発

```bash
npm run dev    # 開発サーバー
npm run build  # ビルド
```
