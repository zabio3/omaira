# おまいらタイピング

```
　 ∧＿∧
　（　´∀｀）  ＜ ﾀｲﾋﾟﾝｸﾞ ﾔﾗﾅｲｶ
　/　　　 ヽ
```

懐かしのネットスラングでタイピング練習するスレ

![Demo](docs/demo.gif)

## 遊ぶ

**[https://zabio3.github.io/omaira/](https://zabio3.github.io/omaira/)**

## 特徴

- 懐かしのネットスラングを収録
- 複数のローマ字入力パターンに対応（shi/si など）
- 単語クリアで +2秒 延長
- タイピング分析（ミス箇所、苦手キー）
- X でシェアして自慢できる

## 開発

```bash
npm run dev    # 開発サーバー
npm run build  # ビルド
```
